Casino Game

This was my second at-home python project. It is a simple game where you bet on a number of lines, and like at a casino if they line up in any row you bet on, the subsequent amount of money is deposited back in your account. If the spin doesnt hit, the subsequent money is taken out of the account. Once the player loses all balance, the game is over and is unable to run anymore.

Discord Python Bot

This Discord Python Bot, my third at-home python project, is a versatile and interactive bot designed to enhance the functionality of Discord servers. It features a variety of functionalities including the ability to play audio from YouTube, send reminders via text messages to specified phone numbers, provide weather updates for specific locations, manage game mods for Cyberpunk and Minecraft, and more. The bot also supports a unique 'Lamester Quotes' feature, allowing users to add and store quotes, which are then periodically sent to a designated channel. Integrated with a MySQL database for storing mod information, and capable of audio playback through FFmpeg, this bot serves as a multifunctional tool for server members, enhancing engagement and interaction within the community.
